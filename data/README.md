# Data Directory

This directory should contain your XAUUSD 1-minute OHLC data files in CSV format.

## Required Files (not included due to size):
- `combined_xauusd_1min_full.csv` - Main 1-minute OHLC data
- `ema200_trend_by_date.csv` - Daily trend data (generated by scripts/generate_ema200_trend.py)

## File Format:
CSV files should have the following columns:
- `timestamp` (UTC timezone)
- `Open`
- `High` 
- `Low`
- `Close`
- `Volume` (optional)
